Como usar (rápido):
1) Instale dependências: pip install iqoptionapi numpy scikit-learn
2) Rode: python3 apptela.py
3) Conecte na conta PRACTICE e clique em 'Iniciar Bot 24h'.

Arquivos principais:
- apptela.py (UI)
- iq_service.py (conexão / candles / buy/check resultado)
- bot_engine.py (bot 100% automático: tendência + pullback + ML gate)
- ml_model.py (modelo online)
- strategy_analyzer.py (confiança mínima por perfil + histórico)
